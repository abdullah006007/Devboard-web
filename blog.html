Different ways to select to select an element in the Dom is : document.getElementById( ), document.getElementsByClassName( ), document.getElementsByTagName( ), document.querySelector( ), document.querySelectorAll( ).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are: - innerHTML allows to insert raw HTML into the document. - innerText updates specific text while ignoring hidden elements. - textContent is like plain text without any HTML formatting.

Question-3: What is event delegation in the DOM?

event delegation is like Instead of adding event listeners to every child element, we add just one event listener to their parent. When a child is clicked, the event moves up to the parent .

Question-4: What is event bubbling in the DOM?

the event moves up to its parent, then to the grandparent, and continues up to the top of the document. This process is called event bubbling in the DOM.
